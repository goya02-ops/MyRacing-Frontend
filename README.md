## ¿Cómo utlizar API MyRacing?

1. Cargar la clase de la entidad a crear con sus respectivos atributos iguales al back en entities.ts

2. Importar la clase y agregar el endpoint en EntityMeta.ts

3. A la función fetchEntities se le pasa por parámetro la clase de la entidad que se quiere buscar

4. A la función saveEntity se le pasa por parámetro la clase y la instancia de la entidad que se quiere guardar
